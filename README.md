# shadowthorn-studios-site
Static website for ShadowThorn Studios. Built with HTML/CSS and deployed via GitHub Pages with custom DNS and HTTPS. Serves as the foundation for future game, AI, and infrastructure projects.

# ShadowThorn Studios

Official website for ShadowThorn Studios.

## Overview

ShadowThorn Studios builds games, AI tools, and digital infrastructure.

This repository contains the static site foundation for the company’s web presence.

## Technical Stack

- HTML5
- CSS3
- GitHub Pages (Static Hosting)
- Custom DNS Configuration
- HTTPS via GitHub SSL

## Purpose

This project demonstrates:
- Static web deployment
- Domain configuration
- DNS management
- Secure hosting setup
- Clean, minimal web architecture

## Roadmap

Layer 1 – Static Site (Current)
Layer 2 – Study Blog (A+ logs, technical notes)
Layer 3 – Project Demos
Layer 4 – AI Tool Landing Pages
Layer 5 – SaaS Infrastructure
